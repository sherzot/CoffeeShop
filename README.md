# CoffeeShop

Zamonaviy, sokin va toza ko‘rinishga ega CoffeeShop landing sahifasi. Loyiha statik HTML/CSS asosida qurilgan va responsive (mobilga mos) dizaynni qo‘llab-quvvatlaydi. Hero bo‘limida yengil video fon, kartalar va menyu bloklari bilan intuitiv, oddiy UX taqdim etiladi.

## Xususiyatlar
- Toza va zamonaviy UI (neutral/cream palitra)
- Hero bo‘limida video fon
- Kartalar, menyu, sharhlar, rezervatsiya va yangiliklar bo‘limlari
- Responsive dizayn (desktop/tablet/mobile)

## Texnologiyalar
- HTML5
- CSS3
- Font Awesome (ikonlar)

## Ishga tushirish
1. Reponi klon qiling.
2. `index.html` faylini brauzerda oching.

## Papkalar
- `index.html` — asosiy sahifa
- `style/` — CSS fayllar
- `images/` — rasmlar
- `videos/` — video fayllar
- `fonts/` — shriftlar

## Deploy
GitHub Pages orqali deploy qilish uchun `main` branchdan foydalaning.
